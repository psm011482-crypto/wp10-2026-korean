# Winning Post 10 2026 한국어 패치

Steam판 Winning Post 10 2026 **버전 1.0.4.0** 전용 비공식 한국어 패치입니다.

> ⚠️ **개발 중인 베타입니다.** 번역과 기술 검수가 끝난 최종판이 아닙니다.
> 일부 일본어, 어색한 번역, 미번역 문장이 남아 있습니다.

## 다운로드

👉 **[Releases](../../releases)**

## 요구 사항

- Steam 정품 Winning Post 10 2026
- **게임 버전 1.0.4.0**
- Windows

다른 버전에서는 설치가 거부됩니다. 설치 시 게임 파일을 검사하기 때문입니다.
**Python 등 별도 프로그램 설치는 필요 없습니다.** 전부 포함돼 있습니다.

## 설치

### 1. 게임 폴더 찾기

Steam 라이브러리 → Winning Post 10 2026 우클릭 → **관리 → 로컬 파일 탐색**

열린 폴더에 `WP10_2026.exe` 가 있으면 맞습니다.

### 2. 폴더째 넣기

압축을 풀면 `WP10_Korean_v0.5.0-beta` 폴더가 생깁니다.
**이 폴더 전체를 게임 폴더 안에** 넣어 주세요.

    Winning Post 10 2026\
    ├── WP10_2026.exe
    ├── DATA.UNB
    ├── FPackages\
    └── WP10_Korean_v0.5.0-beta\   ← 여기
        ├── 1_INSTALL.cmd
        ├── 2_RUN_KOREAN.cmd
        ├── 3_UNINSTALL.cmd
        └── ...

파일을 게임 폴더에 흩어 넣는 방식이 **아닙니다.** 폴더째 넣으세요.

### 3. 설치

1. 게임을 완전히 종료 (작업 관리자에서도 확인)
2. `1_INSTALL.cmd` 실행
3. 관리자 권한 요청에 **예**
4. `Exit code: 0` 확인

원본 파일은 패치 폴더 안 `backup` 에 자동 백업됩니다.
**제거 전까지 패치 폴더와 backup 폴더를 지우지 마세요.**

## 실행

**반드시 `2_RUN_KOREAN.cmd` 로 실행하세요.**

Steam 실행 버튼이나 `WP10_2026.exe` 직접 실행은 안 됩니다.
게임 실행파일에 전자서명이 있어서, 파일을 고치는 대신 실행 중
메모리에 한국어를 넣는 방식이기 때문입니다.

> ⚠️ **게임이 끝날 때까지 CMD 창을 닫지 마세요.**
> 최소화는 됩니다. 강제로 닫으면 번역이 해제되거나 게임이 불안정해집니다.

바탕화면에 `2_RUN_KOREAN.cmd` 바로가기를 만들어 두면 편합니다.

## 제거

게임과 CMD 창을 완전히 종료한 뒤 `3_UNINSTALL.cmd` 실행.
복구 완료와 `Exit code: 0` 을 확인한 **다음에** 패치 폴더를 삭제하세요.

## 적용 범위

| 대상 | 건수 |
|---|---:|
| 대사 (WMM) | 145,512 |
| 메뉴 / UI (EXE) | 11,906 |
| 고정 텍스트 (WCM) | 3,736 |
| 마스터 데이터 (DATA) | 27,549 |

- 기본 UI 및 주요 게임 텍스트
- 경마 히스토리아 장문
- 경마용어사전
- 명마열전 설명
- 한글 폰트

## 알려진 제한

- **말 이름 / 인물 이름** — 미번역
- **한글 이름 직접 입력** — 미지원
- **버튼 이미지 속 글자** — 그림이라 번역 불가
- **번역 품질** — 기계번역 초벌이 상당수. 개선 중입니다
- **전체 구간 실플레이 검수** — 미완료

## 게임 업데이트 / 무결성 검사

다음 작업을 하면 패치가 원본으로 돌아가거나 버전이 바뀔 수 있습니다.

- Steam 게임 파일 무결성 검사
- 게임 업데이트
- 게임 재설치

가능하면 `3_UNINSTALL.cmd` 로 먼저 제거한 뒤 진행하세요.
업데이트 후 버전이 1.0.4.0이 아니면 호환 패치가 나올 때까지 설치하지 마세요.

## 자주 발생하는 오류

| 증상 | 원인 / 해결 |
|---|---|
| `WP10_2026.exe not found` | 패치 폴더 위치가 잘못됨. 게임 폴더 안에 넣으세요 |
| `Unsupported game executable` | 지원하지 않는 버전. 1.0.4.0인지 확인 |
| `Close WP10_2026.exe` | 게임이 아직 실행 중. 종료 후 재시도 |
| 설치는 됐는데 한글이 안 나옴 | Steam으로 실행했을 가능성. `2_RUN_KOREAN.cmd` 로 실행 |
| 보안 프로그램 경고 | 메모리 패치 도구(Frida) 때문입니다 |
